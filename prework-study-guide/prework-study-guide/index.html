HTML

  • HTML is the language which builds a "sturdy frame" for your webpage.
  • It is a good idea to use semantic HTML elements whenever possible to make your code easier to interpret.
  • Declarations tell the browser what doc type it should be when it is opened. While HTML is case sensitive, Declarations are not.

CSS

  • CSS (Cascading Style Sheets) is a language used to define the style properties of a webpage.
  • CSS describes how the elements of an HTML or XML file should appear to a user on the front end.
  • There are three main ways to style a webpage using CSS: inline style sheets, internal style sheets, and external style sheets
  • Out of the main three, external style sheets are most commonly used as they allow you to keep all of your CSS rules in a separate file, making future design changes much easier.
  • The margin property in a CSS sheet defines how much space we want AROUND an element. The margin property is much like using margins in a word document.
  • The padding property adds space AROUND the content INSIDE the element. The element becomes larger when you increase the padding.
